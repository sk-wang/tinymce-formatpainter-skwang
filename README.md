# tinymce-formatpainter-skwang

### Description

A free formatpainter plugin for tinymce

![image-20210304103235202](C:\Users\wangh\AppData\Roaming\Typora\typora-user-images\image-20210304103235202.png)


### Usage

```javascript
tinymce.init({
    selector: '#tinymce',
    plugins: [
        'formatpainter'
    ],
    toolbar: "formatpainter",
});
```
